## Getting Started

### Clone the repository

```bash
git clone https://github.com/Patnala-kousalya/Personal_Portfolio_Website.git
```

### Go to the project folder

```bash
cd Personal_Portfolio_Website
```

### Install dependencies

```bash
npm install
```

### Run the development server

```bash
npm run dev
```

### Build for production

```bash
npm run build
```
